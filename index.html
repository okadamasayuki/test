    端末間の同期設定

    Firebaseの設定がまだありません。リポジトリの README の手順でFirebaseプロジェクトを作成し、firebase-config.js を設定すると同期が使えるようになります。

    ログインすると、メモとファイルがあなた専用の領域に保存され、 スマホやPCなど複数の端末でリアルタイムに同期されます。

    または

    パスワードを忘れた場合（リセットメールを送る）

    としてログイン中。リアルタイム同期が有効です。
    他の端末でも同じアカウントでログインすると繋がります。

    キーはこのブラウザにだけ保存されます。要約の生成にはAnthropicの従量課金（1ファイルにつき数円程度）がかかります。
