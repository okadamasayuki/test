# メモ帳

ブラウザだけで動く、シンプルなメモアプリです。ビルド不要・依存ライブラリなし。

## 使い方

`index.html` をブラウザで開くだけで使えます。

```
open index.html      # macOS
xdg-open index.html  # Linux
```

## 機能

- メモの作成・編集・削除
- 入力すると自動保存（`localStorage` に保存）
- タイトル・本文からの全文検索
- 更新日時の新しい順に自動で並び替え
- ダークモード対応（OSの設定に追従）
- ショートカット: `Ctrl` / `Cmd` + `N` で新規メモ

## 構成

| ファイル      | 役割             |
|---------------|------------------|
| `index.html`  | 画面の構造       |
| `style.css`   | スタイル         |
| `app.js`      | ロジック・保存処理 |

## データについて

メモはブラウザの `localStorage` に保存されます。サーバーには送信されません。
同じブラウザ・同じ端末でのみデータが引き継がれます。
