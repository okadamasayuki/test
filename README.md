# メモ帳

ブラウザで動くシンプルなメモ＆ファイル共有アプリです。
ログインすると、スマホ・PCなど複数の端末でリアルタイムに同期されます。

- 公開URL: https://okadamasayuki.github.io/test/
- `main` ブランチにプッシュすると GitHub Actions が自動でGitHub Pagesへデプロイします

## 機能

### メモ

- 作成・編集・削除、タイトル・本文の全文検索
- 入力すると自動保存
- **対応期日**: なし／本日中／明日中／今週中／来週中／日付指定から選択。
  一覧に期日チップが表示され（期限切れ=赤、本日中=オレンジ）、
  「期日順」ボタンで期日が近い順に並び替えできる
- **ドラッグ＆ドロップで並び替え**: 行の右端のハンドル（⠿）をつかんで移動。
  カードが浮き上がり、他の行がアニメーションで場所を空ける
- 空のまま放置した新規メモは、他の画面へ移った時に自動削除

### ファイル

- 「ファイル」タブで端末間のファイル受け渡し（1ファイル20MBまで、要ログイン）
- **プレビュー**: 画像・PDF・テキスト/JSON・Word(.docx)・Excel(.xlsx)はタップでその場で閲覧。
  プレビューから削除・ダウンロードも可能
- その他の形式（zipなど）はタップでダウンロード
- **AI要約**(任意): ⚙の設定でAnthropicのAPIキーを保存すると、プレビュー上部に
  「AI要約」カードが出る。「要約を生成」を押したときだけClaude APIを呼び、
  結果はFirestoreにキャッシュしてiOSアプリと共有。キーはこのブラウザにのみ保存され、
  生成には従量課金（1ファイル数円程度）がかかる
- xlsxプレビューは1枚目のシートの値のみ。日付はシリアル値を書式から判定して変換するが、
  罫線・色・結合セル・複数シートは扱わない。結果がキャッシュされていない数式は数式そのものを表示

### 共通の操作

- 一覧の行を**左スワイプで削除**（PCはマウスドラッグでも可）
- **☑（選択）ボタンで一括削除**: 複数選択・すべて選択に対応
- スマホではメモをタップすると全画面表示になり、
  **画面の右スワイプ／「← 一覧に戻る」／ブラウザの戻る**で一覧に復帰
- ダークモード対応（OSの設定に追従）、`Ctrl` / `Cmd` + `N` で新規メモ

### アカウント・同期

- ⚙（設定）から **メールアドレス＋パスワードの新規登録**、または
  **Googleアカウントでログイン**（同期中は⚙が緑になります）
- メモ・ファイル・並び順・期日はすべてリアルタイムで他端末に反映
- ユーザーごとにデータは完全に分離され、他人からは見えません
- ログアウトすると端末内のデータは消去されます（クラウドには残り、
  再ログインで復元）
- 未ログインでも、この端末内だけのメモ帳として利用できます
- 設定画面の「サンプルデータを投入」で、お試し用のメモ10件と
  ファイル20件（写真・PDF・Word・Excel を各5件）を一括投入できます
  （すでにあるメモ・ファイルは名前で判定して再投入しません）

## 構成

| ファイル             | 役割                                         |
|----------------------|----------------------------------------------|
| `index.html`         | 画面の構造                                   |
| `style.css`          | スタイル                                     |
| `app.js`             | ロジック・保存・同期処理                     |
| `firebase-config.js` | 同期先(Firebase)の接続設定                   |
| `vendor/`            | 同梱ライブラリ（mammoth.js: docxプレビュー / fflate: xlsxのZIP展開） |
| `samples/`           | サンプルファイル一式                         |
| `.github/workflows/` | GitHub Pagesへの自動デプロイ                 |

ビルド不要の静的ファイル構成です。`index.html` をブラウザで開くだけでも動きます。

## データについて

- メモは常にブラウザの `localStorage` にキャッシュされ、オフラインでも閲覧・編集できます
- ログイン中は Firestore の `users/{あなたのUID}/` 配下（memos / files / chunks）にも保存されます
- 同期はメモ単位の「更新日時が新しい方が勝ち」でマージ、削除は墓標方式で全端末に伝播します
- ファイルはbase64で分割してFirestoreに保存されます（無料枠1GiBの範囲で利用可能）
